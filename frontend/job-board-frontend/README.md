# Job Board Application

A full-stack Job Board web application where users can browse and apply for jobs.

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MongoDB
- Authentication: JWT

## Features
- User authentication
- Job listings
- Apply for jobs
- Recruiter job posting
- Role-based access control

## Setup Instructions

### Backend
```bash
cd backend
npm install
npm run dev
